A JavaScript tone generator

This page uses the Mozilla Audio Data API, present in Firefox 4.0 beta 6 and later. As of this writing (21 Nov 2010) it won't work in other browsers :-(

The ratios between the different pitches that we use in western music is a carefully-worked-out compromise between a number of factors. This has left us in a situation where some chords actually sound nicer when they're slightly out of tune. Try playing the two below and see what you think.

Based on this Firefox Audio Data API demo. Inspiration from this synthesizer 101 post, and, of course, Wikipedia.
