Audio Data API experiments

Some experiments using the new Audio Data API in Firefox 4. As of this writing (21 Nov 2010) they won't work in other browsers :-(

  • A tone generator with attack-decay-sustain-release control.
  • A simple keyboard-based synthesiser.
  • Musical temperament -- fun with music theory.
